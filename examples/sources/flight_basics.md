# 飞行力学基础

## 飞机的基本受力

飞机的基本受力包括升力、重力、推力和阻力。升力是气流作用在机翼上并垂直于相对来流方向的空气动力分量。重力是地球对飞机产生的引力。推力用于克服阻力并维持或改变飞行速度。

## 升力与攻角

升力系数是描述机翼产生升力能力的无量纲指标。攻角影响升力系数，升力系数影响升力。在一定范围内，增大攻角通常会提高升力系数；过大的攻角会导致气流分离，气流分离会导致失速。

升力大小与空气密度、飞行速度、机翼面积和升力系数有关。升力公式为 L = 0.5ρV²SC_L，其中 ρ 表示空气密度，V 表示飞行速度，S 表示机翼面积，C_L 表示升力系数。

## 稳定飞行

平飞是高度和速度基本保持不变的飞行状态。稳定平飞时，升力与重力平衡，推力与阻力平衡。配平用于减小飞行员持续操纵所需的操纵力。
